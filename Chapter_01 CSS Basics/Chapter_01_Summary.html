CSS Tutorial

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML or XML. CSS defines how elements should be displayed on screen, paper, or in other media.

DOM Properties in CSS

The DOM (Document Object Model) represents the HTML document as a tree structure. CSS can be used to style elements in the DOM using:

  • Element selectors (e.g., p)
  • ID selectors (e.g., #header)
  • Class selectors (e.g., .menu)

HTML ID and Class

ID: Used to identify a single unique element. Defined with id attribute.

<div id="main">Content</div>

Class: Used to define a group of elements. Defined with class attribute.

<p class="intro">This is an intro paragraph</p>

How to Add CSS to HTML

There are three main ways to apply CSS:

  1. Inline CSS – within the HTML element using style attribute.
    2. <p style="color: red;">Hello</p>
  2. Internal CSS – inside a <style> tag in the HTML <head>.
    3. <style>
      p { color: blue; }
    </style>
  3. External CSS – in a separate .css file linked via <link> tag.
    4. <link rel="stylesheet" href="styles.css">

CSS Selectors

CSS selectors are used to "select" the HTML elements you want to style.

  • Universal selector: *
  • Element selector: p, h1
  • Class selector: .classname
  • ID selector: #idname
  • Group selector: h1, p, div
  • Descendant selector: div p
  • Child selector: div > p
  • Attribute selector: input[type="text"]

CSS Comments

CSS comments are used to explain the code and are ignored by browsers. They are written like this:

/* This is a CSS comment */
